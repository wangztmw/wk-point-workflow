# Markdown 写作规范

## 基本约定

- 用 `---` 分隔幻灯片
- 用 `<!-- slide: type, key=value -->` 标注类型和属性
- 第一个 h1/h2 自动成为幻灯片标题（除非在注释中显式指定）

## 各类型写法

### title（封面页）
```markdown
<!-- slide: title, theme=gradient -->
# 主标题
## 副标题
```
theme 可选 `gradient`（默认）/ `dark` / `light`。

### toc（目录页）
```markdown
<!-- slide: toc -->
## 目录
### 章节一
### 章节二
### 章节三
```
h3 自动编号。

### section（过渡页）
```markdown
<!-- slide: section -->
# 第一部分
## 副标题（可选）
```

### content（文字页）
```markdown
<!-- slide: content -->
## 标题
- 列表项一
- **粗体强调** 普通文字
- `行内代码`

### 次级标题
1. 有序列表项
2. 第二项
```
支持粗体（`**text**`）、斜体（`*text*`）、行内代码（`` `code` ``）。次级标题用 `###` / `####`。

### summary（总结卡片）
```markdown
<!-- slide: summary -->
## 核心结论
### ✅ 达成目标
- 营收超预期8%
### ⚠️ 需要关注
- 客服响应时间延长
### 🎯 下季度计划
- 拓展海外市场
```
每个 `###` = 一张卡片，列表 = 卡片内容。最多 3 张卡片。

### two-column（两栏布局）
```markdown
<!-- slide: two-column -->
## 标题
### 左栏标题
- 内容一
- 内容二
### 右栏标题
- 内容三
- 内容四
```
列表项均分到两栏。第一个 h3 → 左栏，第二个 h3 → 右栏。

### three-column（三栏卡片）
```markdown
<!-- slide: three-column -->
## 标题
### 栏一标题
### 栏二标题
### 栏三标题
- 内容项
- ...
```
列表项均分到三栏。

### kpi-grid（KPI 概览）
```markdown
<!-- slide: kpi-grid -->
| 指标 | 数值 | 趋势 |
|------|------|------|
| 总营收 | ¥12,800万 | ↑ +18% |
| 活跃用户 | 128万 | ↑ +5% |
| 客单价 | ¥456 | ↑ +8% |
| NPS评分 | 72分 | ↑ +3分 |
```
固定 4 行。第1列=指标名，第2列=数值，第3列=趋势（可选）。

### chart（图表）
```markdown
<!-- slide: chart, type=bar, title=季度销售 -->
| 季度 | 产品A | 产品B |
|------|-------|-------|
| Q1   | 120   | 85    |
| Q2   | 145   | 102   |
```

```markdown
<!-- slide: chart, type=pie, title=市场份额 -->
| 渠道 | 占比 |
|------|------|
| 搜索 | 40   |
| 社交 | 25   |
```

```markdown
<!-- slide: chart, type=waterfall2, title=利润分析 -->
| 阶段 | 金额 |
|------|------|
| 年初 | 300  |
| +收入 | +150 |
| -成本 | -80  |
| 合计 | 370  |    ← 含"合计"自动识别为落地柱
| +增长 | +120 |
| 年末 | 390  |
```

### table（三线表）
```markdown
<!-- slide: table -->
| 季度 | 营收 | 利润 | 利润率 |
|------|------|------|--------|
| Q1   | 580  | 120  | 20.7%  |
| Q2   | 680  | 145  | 21.3%  |
```

### quote（引用页）
```markdown
<!-- slide: quote -->
# 千里之行，始于足下
## 老子
```

### ending（结束页）
```markdown
<!-- slide: ending -->
# 谢谢
## 副标题（可选）

contact@company.com
```

### 图文模板
```markdown
<!-- slide: image-text -->
## 产品展示
![](图片URL或base64)
- 特性一
- 特性二

<!-- slide: image-full -->
# 沉浸式体验
## 副标题
![](背景图URL)

<!-- slide: image-grid -->
## 产品图集
![](图1) ![](图2)
![](图3) ![](图4)
### 标签1
### 标签2
### 标签3
### 标签4
```

## 内联格式

| 写法 | 效果 |
|------|------|
| `**粗体**` | 粗体 |
| `*斜体*` | 斜体 |
| `` `代码` `` | 等宽字体 |
