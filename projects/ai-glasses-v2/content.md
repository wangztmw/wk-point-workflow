<slide: stack; title: 三大件结构：拆开看里面有什么>
<h3: 外结构件（约占总成本20-25%）; font-size: 16>
<list: ; type: bullet>
- 镜框/镜腿：承载所有电子元件的外壳，材质决定重量和质感
- 鼻托/镜片：保证佩戴舒适性，近视用户可配度数镜片
- 类比：相当于汽车的白车身和内饰，是基础承载平台

<h3: 光学显示模组（约占总成本30-50%，如有屏幕）; font-size: 16>
<list: ; type: bullet>
- Micro-OLED/Micro-LED显示屏：米粒大小，发出图像光源
- 光波导/Birdbath光学元件：将显示屏光线引导、放大并投射到人眼
- 类比：相当于汽车的仪表盘和中控屏，是信息呈现窗口

<h3: 电子控制模组（约占总成本30-40%）; font-size: 16>
<list: ; type: bullet>
- AI芯片/主板：眼镜的"大脑"，处理摄像头画面和语音指令
- 摄像头/麦克风阵列：眼镜的"眼睛和耳朵"，感知环境
- 电池/扬声器：提供能源和声音反馈

<img:AI眼镜三大件结构图>

<slide: split; title: 显示技术原理>
<h3: 光波导技术; font-size: 15>
<list: ; type: bullet>
- 像光纤传导，图像经光栅多次反射后进入人眼
- 镜片极薄，外观接近普通眼镜
- 成本高（¥720-200/套），工艺复杂
- 代表：HoloLens、Magic Leap

<h3: Birdbath技术; font-size: 15>
<list: ; type: bullet>
- 像汽车后视镜，图像射向半透半反镜再反射到人眼
- 技术成熟，成本低（¥360-100/套）
- 光学路径长，眼镜整体较厚
- 代表：XREAL Air、Rokid Air

<img:光波导vsBirdbath对比图>

<slide: stack; title: AI技术基础：眼镜怎么"听懂人话"？>
<h3: 端侧AI（眼镜本地运行）; font-size: 16>
<list: ; type: bullet>
- 处理实时任务：唤醒词识别"你好岚图"、简单指令"拍照"
- 响应速度极快（毫秒级），不依赖网络，保护隐私
- 硬件依赖：高通AR1、恒玄BES等专用低功耗AI芯片
- 限制：受限于算力和功耗，无法运行太复杂的模型

<h3: 云端AI（服务器运行）; font-size: 16>
<list: ; type: bullet>
- 处理复杂任务：多轮对话、图像识别、实时翻译
- 算力几乎无限，能运行GPT-4级别大语言模型
- 延迟100-300毫秒，产生流量和云端服务成本

<h3: 多模态AI是当前趋势; font-size: 16>
<p: 同时理解语音（你说的话）+ 视觉（摄像头画面）+ 文字（显示信息），代表：GPT-4o、Mind GPT、澎湃大模型; font-size: 13; color: 555>
