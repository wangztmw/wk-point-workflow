📝 文本可编辑 🎨 SVG → 矢量形状 📊 图表可编辑 ⚡ 纯浏览器端
技术方案核心能力
  • DOM 元素直接映射为 PPTX 原生形状 — 非截图
  • 文本保持可编辑：字体、颜色、大小、粗细全部保留
  • CSS 渐变 → PPT 矢量渐变，阴影 → PPT 极坐标阴影
  • border-radius、RTL 文字、伪元素均正确转换
  • SVG 图形以矢量形式嵌入 → PPT 中可"转换为形状"编辑
  • 100% 浏览器端运行，无需服务器，数据不离开本地
  • 一键导出标准 .pptx，PowerPoint / WPS / Keynote 均可打开
